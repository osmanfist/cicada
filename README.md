# Cicada - Development Task Manager

A lightweight, web-based task management system designed specifically for software development workflows.

## Features
    Yea yea yeah
- **Task Management**: Create, complete, and delete development tasks
- **Priority Levels**: Low, Medium, High, Critical
- **Time Estimation**: Track estimated hours per task
- **Tagging System**: Organize tasks with custom tags (frontend, backend, bug, feature, etc.)
- **Search Functionality**: Find tasks by title, description, or tags
- **Filter Views**: All Tasks, Active, Completed
- **Progress Tracking**: Visual progress bar showing completion rate
- **Local Storage**: Tasks persist in browser between sessions
- **Responsive Design**: Works on desktop and tablet devices
